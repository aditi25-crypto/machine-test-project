# Machine Test Project

This project is built using **NodeJS, Angular, and MySQL (RDBMS)**.

## Features

### Category Master
- Add Category
- View Category List
- Delete Category

### Product Master
- Add Product with Category
- View Product List with:
  - ProductId
  - ProductName
  - CategoryId
  - CategoryName
- Server-side Pagination

## Backend
NodeJS + Express API

## Frontend
Angular Standalone Components

## Database
MySQL

## Pagination Logic
Products are fetched from database based on page number and page size using SQL LIMIT and OFFSET.
